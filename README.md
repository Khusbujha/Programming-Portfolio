This repository is organized to include all work completed during the module Fundamentals of Computer Programming. This repository includes our weekly assignments, logbooks, and project tasks. Below is a brief description of each section:
---
### 1. `notebooks/`
This folder contains **weekly assignments** completed throughout the module. Each notebook corresponds to a specific week and includes the exercises and tasks assigned for that week.
---
### 2. `logbooks/`
This folder includes **learning outcomes** and **weekly summaries** of what we learned during the module. It provides an overview of our understanding and reflections on each week's topics.
---
### 3. `projects/`
This folder contains the **project tasks** assigned to us for the module. This folder contains a `requirements.md` folder stating requirements to deploy each project.
