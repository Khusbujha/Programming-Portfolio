# Python Weekly Assignments

This folder contains the solutions to weekly practical exercises in **.py** format.  
Each weekly exercise includes several programming questions, organized in subfolders representing the corresponding week.

---

## `Week 2`
The tasks cover the use of **user input prompts** and **simple condition statements**.

---

## `Week 3`
Exercises focus on **conditional statements** and **loops**.

---

## `Week 4`
Tasks introduce **functions** and their basic use.

---

## `Week 5`
Exercises involve the use of **modules**, primarily the `sys` module, including some tasks that go beyond the provided syllabus.

---

## `Week 6`
Tasks focus on comprehensive use of **functions, modules, loops, and conditional statements**.

---

## `Week 7`
Exercises emphasize **built-in data structures**, such as lists, tuples, sets, and dictionaries.

---

## `Week 8`
The practical exercises focus on **critical problem solving** and include **file handling operations**.
